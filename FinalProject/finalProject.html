Final Project - Interactive Data Exploration 2017/18

Exploring executions in the United States from 1977 - 2017

Motivation

The following project is the final project of the course Interactive Data Exploration at Diku 2017/18. It has been a collaboration between Aslak Niclasen and Jonathan Kurish.
We've chosen a medium sized dataset which describes all executions performed in the United States from 1977 - 2017. This data is easily available from here and can be downloaded as a .csv file.

Our motivation for choosing this particular dataset is straightforward. We have alot of information from each individual data-component. This gives us the ability to create a more enriching experience because the viewer has a lot of initial control. By doing so, we can smoothly guide and present aspects of the data, which we find interesting. Note the emphasis on we, since we don't which to tell every story possible, but rather enlighten the viewer from our starting point, and let further exploration take over. For the purpose of understanding we start our journey with two main aspects which we want to explore. Firstly: it's interesting to see if certain states are executing more convicted people than others - going further from here whats their race. This could be due to Federal laws in the United States being different from state to state. As an example, some states doens't have death penalties at all. Secondly: we wish to see if convicts whom killed more than 1 person is executed differently. That is, are we using a particular type of execution for multiple homocide convicts. With these questions in mind, we now describe dataset.

The dataset

The dataset consists of 1467 individual rows each corresponding to an execution. Each execution is labeled with a set of distinct features of which the most important are listed below. Note that two rows in the dataset did not contain any Date of execution, and has therefore been discarded. Leaving us a total of 1465 executions.

  • Date of execution
  • Name of the convicted
  • Age
  • Sex
  • Race
  • Number of victims | Their race | Their sex
  • State of crime incident and execution
  • Region of crime incident within state
  • Race of victim(s)
  • Type of execution

Overview of all executions

Following the introduction above we now present an overview of the entire dataset. Each execution is marked as a circle where the radius of the circle represent the number of victims, and the color of the circle represent the Type of execution. Each circle can be hovered, which in return shows additional information on that execution. By adjusting the timeline slider below the graph it's possible for the viewer to see the recorded executions on a yearly basis. The full x-axis represents a single year from 01/01/yyyy - 31/12/yyyy. The y-axis marks the age of the convicted. If one wishes to see the entire 30 year period (all datapoints at the same time), it's possible to switch from a yearly view to a full view instead. If marked the slider is set as inactive and the circles become clickable instead of hovering.

For even more information, the viewer has the opportunity to toggle different features on and off on both timeline views. As an example consider picking only executions where the type of execution was Lethal Injection. Later we will discuss interesting types of correlations found with the introductary questions in mind.

Pick a year

Below is a histogram of each individual type of execution and it's total number of occurences. As seen (when hovering), the most dominant type is Lethal Injection with 1290 records, followed by Electrocution at 158, Gas Chamber at 11, closing at Hanging and Firing Squad both at 3. Thus Lethal Injections has been used about 8 times more than Electrocutions. This finding is important because it shows that certain types of executions are very rare. In fact, the very harsh types of execution (excluding Electrocution) account for only ~1.1%. It might again be due to US laws requiring the convicted should not suffer unnecessary during the execution. From the timeline view we can especially see that Lethal injections are dominating the later years, thus supporting this find. Further more this dataset only consist of data post 1977, which is why this may be a bit scewed. We would probably see a larger extend of hanging and firing squad in earlier years, where the Lethal Injection substances used weren't discovered at that time.

In order to find out if the number of victims influence the type of execution, we need to correlate these variables. To do so we present a scatterplot. As can be seen, the two axis represent the two components: The type of execution (x-axis) and the number of victims (y-axis).
